# WEBBTNHOM7 - Hệ thống quản lý trường học

## Mô tả
Hệ thống quản lý trường học với chức năng Báo cáo tổng hợp điểm tốt nghiệp.

## Cấu trúc
- `/server` - Backend API (Node.js + Express)
- `/public` - Frontend (HTML/CSS/JS)

## Chạy project
```bash
cd server
npm install
npm start
```

Truy cập: http://localhost:3000

## Tính năng
- Báo cáo tổng hợp điểm tốt nghiệp theo khóa học
- Xuất báo cáo Excel/PDF
- Lọc theo ngành học, khóa học
